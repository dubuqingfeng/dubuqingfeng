  1. 电源与共地（先保证供电不掉压、不会把 MCU 拉挂）
  2. 串口链路（PC/Android -> 蓝牙/USB-串口 -> MCU/舵机控制板）
  3. 舵机单元测试（单舵机 -> 多舵机）
  4. 机械装配与限位（推进、夹持、旋转的重复精度）
  5. 整机联调（按动作序列执行，遇到异常可停机松手）
